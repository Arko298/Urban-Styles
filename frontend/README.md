## Getting Started

Follow these instructions to set up the project on your local machine.

### Prerequisites

- **Node.js** (v14 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**:

   ```bash
   git clone <repository-url>
   cd urban-styles
   ```

2. **Install dependencies**:
   npm install

3. **Start the development server**:
   npm run dev
